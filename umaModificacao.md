# Sobre o Git

## O que é o Git?
O git é uma ferrementa de vecionamento de código, siginifica que
é uma ferramenta que nos ajuda a criar e admnistrar diferentes 
versões do nosso código, com ela podemos trabalhas em uma "linha do
tempo principal", voltar a uma versão anterior do nosso código e 
trabalha isoladamente em um trecho de código.
